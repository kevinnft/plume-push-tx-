
**Install Script**
```
git clone https://github.com/kevinnft/plume-push-tx-
cd discord-plume-push-tx-

```
python3 -m venv dc
source dc/bin/activate
```
**install bahan**
```
pip3 install -r requirements.txt
```
**Mainkan script**
```
python3 bot.py
```
